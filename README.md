## 🚀 راهنمای نصب

1. پوشه **htdocs** موجود در فایل دانلود شده را با پوشه **htdocs** در محل نصب **XAMPP** جایگزین کنید.
2. برنامه **XAMPP** را اجرا کرده و سرویس‌های **Apache** و **MySQL** را Start کنید.
3. مرورگر خود را باز کرده و وارد آدرس زیر شوید:

```text
http://localhost/pricelist
```

اکنون وب‌اپ آماده استفاده است.
